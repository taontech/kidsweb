# kidsweb 儿童浏览器

    我是一名家长，随着孩子的成长，我深知网络对一个孩子的影响力有多大，
    目前的网络环境噪音太多，各种不健康的东西充斥着各个页面，
    我觉得有责任实现一个优质儿童网站的推荐app，让家长可以放心的给孩子使用。
    第一步提供不健康内容过滤，护眼提醒，安全浏览等功能。
    第二步实现智能推荐
